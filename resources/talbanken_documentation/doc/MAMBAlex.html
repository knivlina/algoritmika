Lexical categories in MAMBA

Lexical categories in MAMBA consist of a two-character part-of-speech tag, optionally followed by up to three two-character tags indicating additional features (mainly morphosyntactic properties). The following table lists all part-of-speech tags, as well as feature tags that can cooccur with a particular group of part-of-speech tags.
